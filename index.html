cloudydict

Cloudydict is a simple python module that makes accessing cloudfiles as easy as using a dictionary. With cloudydict, gone are worries about proprietary APIS and complicated refactorings to migrate.

Installation

Its on Pypi. Just run this:

$ pip install --upgrade cloudydict

Tutorial

Assuming you've setup your .boto configuration file, working with S3 and cloudydict is as simple as:

>>> from cloudydict import cloudydict         
>>> d = cloudydict('s3:cloudydict-demo')   
>>> d['a'] = 'Hello world'         
>>> d['b'] = 'How are you?'          
>>> d.keys()         
>>> d.values()
